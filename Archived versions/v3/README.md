# Archive v3

This folder preserves version 3 ontology materials.

## Files

| File | Role |
| --- | --- |
| `ActInf Ontology version 3.xlsx` | Spreadsheet source in XLSX format. |
| `ActInf Ontology version 3.ods` | Spreadsheet source in ODS format. |
| `ActInf Ontology version 3.zip` | Packaged release artifact. |
| `ActInf Ontology version 3 - Defintions.tsv` | Definitions TSV retained with its original filename. |
| `ActInf Ontology version 3 - Translation.tsv` | Translation TSV. |

## Provenance Notes

- Release date is not encoded in this folder name.
- The misspelling in `Defintions.tsv` is preserved because it is part of the
  historical filename.
- Treat these files as immutable historical source evidence.
